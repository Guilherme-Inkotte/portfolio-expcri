Portfólio - Experiência Criativa: Navegagndo na computação

1

Seek in Space

O primeiro projeto desenvolvido na disciplina foi o jogo Seek in Space, um bom e velho spaceshooter. Com a ferramenta Construct3, fomos desafiados a construir o jogador, os inimigos, o ambiente, a trilha sonora e muito mais. A experiência final ficou bem legal, criamos até um chefão, que ficou bem desafiador.

Grupo: Gabriel Marchioro Klein, Guilherme Henrique Schneider Inkotte, Lucas Ricardo Nagano Rigon e Rodrigo Alves Bolincenha

Tecnologias utilizadas:
Construct3
2

Player de Música

O segundo projeto que ajudei a desenvolver foi o Player de Música. Com o tema de aplicativo multimídia, decidimos fazer um aplicativo de streaming de música, semelhante ao Spotify. Criei o layout no Adobe Illustrator e trabalhamos audio e vídeo num aplicativo visualmente bonito e funcional.

Grupo: Enzo Ramon Campa, Guilherme Henrique Schneider Inkotte e Paula Beatriz Louback Jardim

Tecnologias utilizadas:
Processing Java
3

Site de turismo

O terceiro e último projeto foi um aplicativo Web. Desenvolvimento Web é a parte que eu mais gosto e conheço, então pudemos fazer um site bem interessante. Imagens de locais bonitos e interessantes para se visitar e vídeos de turismo em cidades conhecidas resumem o que apresentamos com o projeto 3. Além disso, um cadastro em PHP para o banco de dados.

Grupo: Gabriel Marchioro Klein, Guilherme Henrique Schneider Inkotte e Rodrigo Alves Bolincenha

Tecnologias utilizadas:
HTML CSS JavaScript PHP MySQL
